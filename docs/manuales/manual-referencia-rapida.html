Acceso al Sistema

  • URL: http://localhost:8000/admin
  • Inicio de sesión: usuario y contraseña proporcionados por el administrador
  • Cerrar sesión: menú superior derecho → nombre de usuario → Cerrar Sesión

Crear actividad (Asistente)

  1. Actividades → Crear con Asistente
  2. Paso 1: Información General (usuario, fecha, oficina, programa, estado)
  3. Paso 2: Detalles (macroactividad, lugar)
  4. Paso 3: Fechas (inicio, vencimiento, recordatorio opcional)
  5. Paso 4: Archivos adjuntos (máx. 10 archivos, 10 MB c/u)
  6. Click en Crear Actividad

Añadir Rápido

Actividades → Añadir Rápido → llenar campos → crear

Editar

Icono lápiz ✏️ (no disponible si está Completada, Cancelada o mes cerrado)

Completar

Icono check verde ✓ → ingresar asistentes → Confirmar

Adjuntar archivos

En edición/vista, pestaña Atestados. Arrastrar o seleccionar. Tipos: PDF, Word, JPG, PNG.

Comentarios

Pestaña Comentarios en la vista de actividad. Solo el autor puede editar/eliminar.

Cierres Mensuales

  • Individual: una departamental, un mes. Bloquea actividades.
  • Consolidado: todas las departamentales, un mes. Solo SuperAdmin/GOL.
  • Estados: Generado → Aprobado → (opcional) Reabierto
  • % Cumplimiento = Ejecutadas / (Ejecutadas + Pendientes)
